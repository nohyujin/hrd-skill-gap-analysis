# 🎯 HRD 스킬 갭 분석 대시보드

Python과 Streamlit을 활용한 **HR 스킬 갭 분석 및 시각화 도구**입니다.

조직 내 직원들의 현재 역량과 목표 역량 간의 차이를 시각적으로 분석하고, 데이터 기반의 교육 계획 수립을 지원합니다.

## ✨ 주요 기능

- 📊 **인터랙티브 레이더 차트**: 현재 역량과 목표 역량의 시각적 비교
- 📈 **스킬 갭 분석**: 보완이 필요한 역량 자동 식별
- 🎨 **직관적인 UI**: Streamlit 기반의 사용하기 쉬운 웹 인터페이스
- 💾 **간편한 데이터 관리**: CSV 파일 기반 (별도 DB 불필요)

## 🛠️ 기술 스택

- **Python 3.x**: 백엔드 로직 및 데이터 처리
- **Streamlit**: 웹 대시보드 프레임워크
- **Pandas**: 데이터 핸들링
- **Plotly**: 인터랙티브 시각화

## 📦 설치 방법

### 1. 저장소 클론
```bash
git clone https://github.com/nohyujin/hrd-skill-gap-analysis.git
cd hrd-skill-gap-analysis
```

### 2. 필수 패키지 설치
```bash
pip install streamlit pandas plotly
```

또는 requirements.txt 사용 (추후 제공):
```bash
pip install -r requirements.txt
```

## 🚀 실행 방법

```bash
streamlit run app.py
```

브라우저가 자동으로 열리며 대시보드가 표시됩니다. (기본 주소: http://localhost:8501)

## 📋 데이터 형식

`skills_data.csv` 파일을 다음 형식으로 준비하세요:

```csv
사원명,직무,스킬 항목,현재 점수 (Actual),목표 점수 (Target)
김철수,데이터 분석가,Python,3,5
김철수,데이터 분석가,SQL,4,4
김철수,데이터 분석가,통계,2,4
이영희,HRD 담당자,기획력,4,5
이영희,HRD 담당자,커뮤니케이션,5,5
```

**필드 설명:**
- **사원명**: 직원 이름
- **직무**: 직무/직책
- **스킬 항목**: 평가할 스킬명
- **현재 점수**: 1-5 척도의 현재 역량 수준
- **목표 점수**: 1-5 척도의 목표 역량 수준
